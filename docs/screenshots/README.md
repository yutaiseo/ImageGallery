## 📝 截图规范

- **分辨率**: 1920x1080 或更高
- **格式**: PNG（保真）
- **大小**: 单张 < 500KB（使用压缩工具）
- **内容**: 
  - 使用演示数据（隐藏真实信息）
  - 界面整洁美观
  - 功能展示清晰

## 🎨 美化建议

1. 使用浏览器开发者工具调整为统一尺寸
2. 确保有足够的演示内容（图片、数据）
3. 隐藏个人敏感信息（用户名、域名等）
4. 可使用图片编辑工具添加边框/阴影

## 📦 截图工具推荐

- **Windows**: Snipping Tool / Snagit
- **macOS**: Command + Shift + 4
- **浏览器**: 全页面截图插件
- **压缩**: TinyPNG / ImageOptim
